lgb.get.eval.result {lightgbm}R Documentation

Get record evaluation result from booster

Description

Get record evaluation result from booster

Usage

lgb.get.eval.result(booster, data_name, eval_name, iters = NULL,
  is_err = FALSE)

Arguments

booster

Object of class lgb.Booster

data_name

name of dataset

eval_name

name of evaluation

iters

iterations, NULL will return all

is_err

TRUE will return evaluation error instead

Value

vector of evaluation result

Examples

## Not run: 
library(lightgbm)
data(agaricus.train, package = "lightgbm")
train <- agaricus.train
dtrain <- lgb.Dataset(train$data, label = train$label)
data(agaricus.test, package = "lightgbm")
test <- agaricus.test
dtest <- lgb.Dataset.create.valid(dtrain, test$data, label = test$label)
params <- list(objective = "regression", metric = "l2")
valids <- list(test = dtest)
model <- lgb.train(params,
                   dtrain,
                   100,
                   valids,
                   min_data = 1,
                   learning_rate = 1,
                   early_stopping_rounds = 10)
lgb.get.eval.result(model, "test", "l2")

## End(Not run)
